Math

timeStillNeeded = Σ max(0, budget − spent) for non-completed cats
timeLeft = secondsUntilDayEnd
slack = timeLeft − timeStillNeeded
used = Σ min(spent, budget) · planned = Σ budget

slack ≥ 0 → On track, sub = "+Xh slack"
slack < 0 → Behind, sub = "−Xh"
all done → Complete, sub = "Xh under budget"
